# Systematic-Map-Agronomic-Research-and-Climate-Change-in-Mediterranean-Type-Climate
This repository contains R code used for the systematic mapping published in [Buchanan, K et al. 2025. Cultivating Climate-Smart Crop Systems: A Systematic Map of Agronomic Interventions in a Mediterranean-Type Climate.]

The repository includes the following: 
  data : the raw data in an Excel worksheet called "Database" in the file: Supplementary Material V2 
  R.scripts : the r code used for the analysis
  Main.R : the full r code in one scipt
